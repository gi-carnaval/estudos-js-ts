## Como Executar o Projeto

1. **Clone o repositório:**

    ```bash
    git clone https://github.com/gi-carnaval/estudos-js-ts.git
    ```

2. **Instale as dependências:**

    ```bash
    cd estudos-js-ts
    npm run start
    ```
**Os scripts serão executados no terminal**
